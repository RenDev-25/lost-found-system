# Mini University Lost & Found Web-Based System

A web application to help university students and staff report, search for, and recover lost items on campus.

## Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Frontend   | React.js + TypeScript + Tailwind  |
| Backend    | Laravel (PHP)                     |
| Database   | MySQL                             |
| Auth       | Laravel Sanctum                   |
| Deploy     | Vercel (frontend) + Railway/Render (backend) |

## Project Structure

```
lost-and-found-system/
├── frontend/    React + TypeScript + Tailwind
├── backend/     Laravel REST API
├── database/    SQL schemas and seed data
└── docs/        Documentation
```
