# System Overview

## About
The Mini University Lost & Found Web-Based System helps students and staff
report, search, and recover lost items on campus.

## Tech Stack
- Frontend: React.js + TypeScript + Tailwind CSS
- Backend: Laravel (PHP) REST API
- Database: MySQL
- Auth: Laravel Sanctum

## Access Control
- System is restricted to verified university accounts only.
- Roles: student, staff, admin
